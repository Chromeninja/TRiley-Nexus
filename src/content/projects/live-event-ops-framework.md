summary: Created a repeatable end-to-end framework for managing large-scale live gaming events, from concept to execution to post-event retrospective, serving events with 200-800 concurrent participants.
problem: Each event was rebuilt from scratch, leading to inconsistent quality, missed details, and exhausted coordinators. There was no institutional knowledge capture or standardized playbook.
approach: Conducted retrospectives across 12 past events, identified failure modes, and built a tiered event playbook system covering pre-event planning, live operations, contingency handling, and post-event analysis.
outcome: Events using the framework had measurably fewer critical incidents, faster issue resolution, and consistent post-event satisfaction scores above 90%. The playbook was adopted by three additional event teams.
